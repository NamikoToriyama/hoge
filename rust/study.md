# Study Rust
Study-memo of Rust.

Link: https://doc.rust-jp.rs/book-ja/ch01-02-hello-world.html

## Hello world
Compile and execute.
```
$ rustc main.rs
$ ./main
Hello, world!
```

## Cargo
Create new project
```
$ cargo new hello_cargo --bin
```

Build and execute
```
$ cargo build
$ ./target/debug/hello_cargo 

or

$ cargo run
```
